# 🐟 FishAI Live Smart Selling System

An AI-powered fish price prediction and smart selling recommendation web application built using Python, Streamlit, Machine Learning, and Plotly.

## 🚀 Live App
https://fishai-live-app-mrmjcwnsi6q5t92eciq5k3.streamlit.app/

---

## 📌 Project Overview

This system analyzes fish market conditions and predicts fish prices based on:

- Location
- Fish Type
- Demand Level
- Supply Level
- Quantity Sold
- Competitor Price

The AI system then recommends whether sellers should:

- SELL NOW
- WAIT
- MONITOR

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
